# U2bdl-GUI
GUI for youtube_dl Project
使用 nw.js 实现的 youtube_dl 界面

------

运行：

- [x] 获取 nw.js
- [x] 将 nw.js 加入环境变量
- [x] 自行获取 youtube_dl.exe 和 ffmpeg.exe
- [x] cd U2bdl-GUI
- [x] 将上述两个文件放置于 ./src/lib/ 路径下，如果没有 lib 自行创建
- [x] npm install
- [x] npm run start

------

打包：

- [x] cd src
- [x] npm install
- [x] cd ..
- [x] npm run build
